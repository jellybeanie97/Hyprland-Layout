# **RICE CONFIG FOR THE FOLLOWING ENVIRONMENTS**

- **Hyprland**
- **Sway**
- **KDE plasma**
- (Others will be added soon..)
  
## **Hyperland**

- Notice how there are 2 folders. One is for the hypr layout and GUI format. The other is for the hyprpanel bar config.
  - Hypr has all the necessary files you need to have the skeleton foundation of the hyprland setup.
  - Hyprpanel has all the bar settings and they are customizable to your liking.

***

## **Sway**

- The sway folder contains the base skeleton layout for the GUI.
- As far customizing the terminal that will be under Kitty.
- Wofi will contain the files on how to access your applications.
- Then added yay to install packages or applications through the terminal.
- The eww folder contains the scripts for wifi interaction and bluetooth. It holds the style of the icons on the waybar.
- Waybar folder contains the style for the notification bar/menu bar.
